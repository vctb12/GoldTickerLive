# Verification Checklist

## Functional
- Buttons work
- Filters work
- Deep links work
- State persistence works

## UX
- Mobile layout checked
- Empty/loading/error states checked
- Contrast checked

## SEO
- Title and meta description
- Canonical
- Sitemap entry
- Internal links
- Structured data where relevant

## Trust
- Freshness labels accurate
- Spot vs retail distinction clear
- Estimated/derived values disclosed
