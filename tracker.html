Methodology + sources

This page is intentionally explicit about what is live, what is cached, and what is only a market-linked estimate.

Live spot layer

Live spot is pulled from Gold API. The site uses USD/XAU as the base market reference and derives local estimates from that stream.

Gold API docs

FX conversion layer

FX conversion comes from the open ExchangeRate-API endpoint. AED still uses the official peg in the repo’s shared constants.

ExchangeRate-API docs

Historical layer

Recent sessions use local snapshots. Longer windows fall back to public historical baselines so charts keep working instead of going blank.

DataHub gold dataset

News wire

The live wire uses the GDELT DOC API in article-list mode with a recent rolling timespan, which is suitable for static-site headline strips.

GDELT DOC API
