Methodology and source stack

Strong trackers explain themselves. This section is built into the page so users understand what the numbers mean and where they do not.

Transparent by design

Live layer

The page fetches live gold spot data in USD and treats it as the market reference per troy ounce. Currency conversion is then applied to local estimates.

Gold API docs Live XAU endpoint

FX layer

Local prices use open exchange-rate data from USD. Because this endpoint updates daily, the page caches rates and uses them for estimation rather than pretending they are tick-level FX prices.

ExchangeRate API docs Open USD rates endpoint

Historical layer

The page first tries a daily-updated historical feed with CORS support, then falls back to a monthly public dataset if needed. That improves short-range charts without pretending to have tick-level archival data.

FreeGoldAPI DataHub fallback dataset

Local estimate formula

Selected price = live spot per oz × purity factor × unit conversion × FX rate. It is a bullion-equivalent estimate, not a guaranteed retail jewelry quote.
