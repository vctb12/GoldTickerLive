Is this the shop price?

No. Gold Ticker Live shows spot-linked reference estimates. Shop prices can add making charges, premiums, taxes, and seller margins.

⏱️

How fresh are prices?

Live pages show freshness labels. If a source is delayed or cached, the interface should label it instead of pretending it is live.

🇦🇪

Why is AED special?

AED gold conversions use the official fixed peg of 3.6725 AED per USD, so UAE prices move mainly with XAU/USD rather than FX noise.

💎

Why do karats change the price?

Karat is gold purity. 24K is near pure gold; 22K is 91.7%; 21K is 87.5%; 18K is 75%. The reference value scales with purity.
