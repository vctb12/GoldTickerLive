ℹ️ About Automation

Posts are automatically generated and posted every hour by the GitHub Actions workflow gold-price-tweet.yml. The template rotates based on Dubai time (UTC+4):

  • Special templates fire at fixed hours: 00:00 (Daily), 06:00 (Morning), 08:00 (Arabic), 10:00 (Country), 14:00 (Comparison), 16:00 (Educational), 20:00 (Evening)
  • Weekend edition replaces the noon post on Fridays & Saturdays
  • All other hours use the Hourly Snapshot template

Templates are read-only — they are defined in scripts/tweet-gold-price.js. To modify a template, edit that file and push to main.

Use the Post Generator tab to manually create and post custom content at any time.
