## Reproduce

```bash
# node cleanup-audit (reachability graph + buckets + dupe detection)
node scripts/node/cleanup-audit.js
# writes /tmp/audit/*.json; copy the ones you want committed into this dir.

# supplementary tools
npx depcheck --json > depcheck.json
npx knip --reporter json > knip.json
# purgecss is scripted inline; see the command used in the corresponding PR.
ruff check --select F401,F811,F841 scripts/python/ --output-format=json > ruff.json
```
